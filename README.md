# UI Testing Example

Source code for the UI Testing blog post.

It uses Maven, Apache Wicket, TestNG and Selenium to present a demo application that is then tested via a Selenium WebDriver.

###To try it out

cd application

mvn clean install

mvn jetty:run

cd ..

mvn clean install
